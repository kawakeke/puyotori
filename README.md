# しりとりハント3D

**[▶ ここで遊べます](https://kawakeke.github.io/puyotori/)** — インストール不要・PC/スマホ対応・90秒で1ゲーム

大人のための、早い者勝ち3Dしりとり。

---

## これは何か

しりとりを、**ターン制の言葉遊びから「早い者勝ちの狩り」に作り変えた**ゲームです。

あなたは3D空間の真ん中に立っています。周囲360度、頭上から足元まで、無数の言葉が魚のように漂っています。画面に出るお題は一文字だけ ——「り」。あなたは空間を見回し、「り」で始まる言葉を探し、網で捕まえる。捕まえた言葉の最後の文字が次のお題になり、しりとりが進みます。

ただし、同じ言葉を狙っているライバルのUFOがいます。あなたが迷っている間に、相手は答えを持っていきます。**考える時間と、動く時間が、同じ時間軸の上にある** —— これがこのゲームの核心です。

そして最大の仕掛けが「ん」です。しりとりで「ん」を言ったら負け、というルールを、このゲームは**人生の落とし穴**として翻訳しました。空間に漂う「誤送信」「終電」「左遷」「借金」「割り勘」——どれも魅力的な高得点に見えますが、掴んだ瞬間にすべてが終わります。捕まえたい気持ちと、捕まえてはいけない理性のせめぎ合いが、しりとりのルールと大人の悲哀をひとつに重ねます。

## 基本ルール

- 制限時間は **90秒**。あなたと相手(UFO)のスコアを競います
- **ドラッグ**で空間を見回す。答えは大抵、いま見ていない方角にあります
- お題の文字で始まる言葉を捕まえる。捕獲方法は2つ:
  - **長押し**(0.65秒)—— 網がすぼまって捕らえる
  - **ダブルクリック(タップ)したまま、ぐるっと囲む** —— 投げ縄。一筆書きで連続捕獲も可能
- 得点は **文字数 × 5点**
- **「ん」で終わる言葉を捕ると -15点、2秒動けなくなる**

## ゲームシステムのポイント

### 1. 長い言葉ほど高得点。だが、小さく、速く、逃げる

「もち」は10点、「まえむきにけんとう」は45点。ただし長い言葉ほど**表示が小さくなり、空間を速く漂います**。さらに全ての言葉は、**網が近づくと震えて逃げます**——長い言葉ほど、すばしっこく。

高得点は文字通り捕まえにくい。この一貫した設計により、「安全に小魚を積むか、大物を追うか」という選択が常に発生します。

### 2. 「ん」トラップは、字面では見抜けない

「ぼうねんかい」は安全ですが「ごうこん」は罠。「檸檬(れもん)」は罠ですが「蓮根(れんこん)」も罠。読めても助かりません。

唯一のヒントは、**罠の言葉が時々かく冷や汗(💦)**です。観察している人だけが気づく、控えめなテル(手癖)として仕込んであります。

### 3. 相手(UFO)の性格が難易度になっている

難易度は速さの調整ではなく、**相手の思考の変化**として実装されています。

| 難易度 | 相手の狙い |
|---|---|
| **接待(易)** | 安い言葉から取る。高得点をあなたに残してくれる |
| **定時(並)** | 手当たり次第 |
| **激務(難)** | 高得点を狙い撃つ。あなたが狙っている大物を、横から掻っ攫う |

相手はトラクタービームで言葉を吸い上げます。ビームが伸び始めたら、その言葉はもう間に合わないかもしれません。

### 4. コンボと、締切効果

連続で捕まえるとコンボボーナス(+5ずつ、最大+25)。獲得音が一段ずつ高くなっていくので、耳でも連鎖が分かります。

そして**残り10秒で全得点が2倍**になります。締切前だけ本気が出る、あれです。終盤の逆転が構造的に起きるうえ、相手の得点も2倍になるため、最後の10秒は取り合いになります。

### 5. 3つのことばセット

| セット | 語数 | 罠 | 中身 |
|---|---|---|---|
| 🍎 **定番** | 215語 | 26語 | りんご、ぺんぎん、ろけっと。素直に遊べます |
| 難 **漢字** | 215語 | 23語 | **蒲公英・啄木鳥・心太**。読みでしりとりするので、読めないと繋がりません。捕まえた瞬間に読みが判明します |
| 🍺 **ビター** | 214語 | **43語** | 残業、二日酔い、半額シール、リボ払い。**5語に1語が罠**という、人生の縮図みたいな確率です |

漢字セットは「饂飩(うどん)」のように、**読めても罠**という二段構えになっています。

### 6. 邪魔をしない演出

4秒間なにも操作しないと、答えのある方向の**画面の隅がほんのり金色に明滅**します。矢印も文字も出ません。気づかない人は気づかないまま、遊べます。

## トーンについて

UI文言はすべて**真顔**です。難易度は「接待/定時/激務」、獲得は「回収」、罠を踏むと「これだから月曜は。」「報告書は不要です。」と真顔で慰められます。結果画面ではスコア差に応じた講評が出ます——完敗すると「UFOにも負ける日があります。今日がその日でした。」

メイドインワリオやせがれいじりのような、**ビターでシュールな笑い**を狙っています。にやりとしてもらえたら成功です。

## 技術メモ

- **依存ライブラリなし・ビルド不要の単一HTMLファイル**(`index.html`、約200KB)。オフラインでも動きます
- 3D描画は**自作の生WebGL**。three.js等は不使用
  - **WebGL層**(世界): 言葉のビルボード、距離フォグ、ライティング付きUFOメッシュ、加算合成のグロー、ワイヤーフレームの床グリッド
  - **Canvas 2D層**(UI): 網カーソル、投げ縄、メッセージ、トラクタービーム
  - 両層は**厳密に同一の投影計算**を共有しているため、当たり判定と見た目が完全に一致します
- 言葉は方位角・仰角・距離の球面座標で管理。テクスチャは絵文字/漢字+ラベル+点数バッジを256px canvasに焼いて生成
- 効果音はWeb Audio APIで合成(音声ファイルなし)
- 通算成績は localStorage に保存
- `puyoritori.html` は初期に作った別ゲーム「ぷよりとり」(ぷよぷよ×しりとり)です
